About Family

My name is Austin Iheke Jr., and I created this website using HTML and CSS as a personal project. It’s designed to showcase my family and some of the things I’ve learned while practicing web development. Each section of the site highlights a different family member with a simple stick figure illustration and a short description. The layout, styles, and code were all built by me from scratch to help me improve my design and coding skills. This project is a combination of creativity and learning, and it serves as a basic example of how personal topics can be turned into web content using front-end development tools.
